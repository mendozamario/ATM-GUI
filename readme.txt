Este es un algoritmo elaborado con C# y WindowsForms que permite calcular
la cantidad de billetes de cada denominación que deberia dar un cajero
automatico.

Para la ejecución de este es necesario tener como sistema operativo Windows
y contar con Visual Studio instalado, solo es necesario clonar el repo y
abrirlo con Visual Studio, solo faltaria compilar y ejecutar.


Programa elaborado por Mario Mendoza Rodriguez - Estudiante de Ingenieria de
Sistemas en la UPC. 2023
